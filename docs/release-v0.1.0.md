# v0.1.0

Initial public release of MCP to OpenAI Tool Schema Lab.

## Included

- Offline CLI for converting MCP `inputSchema` fixtures to OpenAI-style `type: "function"` tool schemas.
- Offline validator with JSON and markdown output.
- Sample MCP tools with `outputSchema` metadata.
- Expected OpenAI-style output fixtures.
- Setup, schema notes, UTM link docs, issue templates, MIT license, and local check script.

## Scope

This release is a schema playground/test fixture repository. It is not a full agent framework, hosted proxy, or production compatibility guarantee.
